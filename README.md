# 圖學P1
## 說明
* 基本上就是一個圖片處理的project，而裡面圖片處理的地方都是自己寫，並非直接拿別人寫好的功能來call
## 檔案執行
* 點開資料夾P1裡面的P1.exe即可執行
## 執行畫面
一點開後會彈出此畫面，請選OpenGL
